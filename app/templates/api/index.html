University Reservation System API

The University Reservation System is an online tool that helps you reserve spaces and equipment at your university. It gives you all the information you need about the thing you are reserving, such as condition, amenities, and more, to ensure that you get exactly what you are looking for.

This API is for anyone who is interested in the space and equipment data we have. This documentation will teach you how to get data regarding the current spaces, equipment, and reservations at your university and also how to add new spaces and equipment for others to use.

Receiving Data

In this section, you will learn how to perform basic querying operations to get the information we have in our database.

Sending Data

In this section, you will learn how to perform operations used to send information to our database.

POST /api/spaces

Adds a new space to the database.

Example

Request

curl -H "Content-Type: application/json" -X POST -d '{"capacity":20,"name":"A room"}' urs-tufts.herokuapp.com/api/spaces

Response
{
  "capacity": 20, 
  "condition": null, 
  "created_by": null, 
  "date_created": "2017-04-20T02:55:34.091498", 
  "id": 23, 
  "last_updated": "2017-04-20T02:55:34.091509", 
  "last_updated_by": null, 
  "location_id": null, 
  "name": "A room", 
  "space_type_id": null
}
